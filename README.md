Webinar https://geekbrains.ru/events/631

## Build
### Gradle
gradle clean build

### Maven
mvn clean package

## Run
### Gradle
gradle bootRun

### Maven
mvn spring-boot:run

## Swagger
http://localhost:9000/quotes/swagger-ui.html

## Dependencies
- junit
- swagger
- spring.boot
